# DAF1 Drift 🚗🔥

DAF1 Drift is a futuristic Gulf cyberpunk open-world racing game powered by Web3 and the DAF1 ecosystem.

Players can:
- Drift and race in a futuristic city
- Upgrade and customize cars
- Earn and use DAF1 coins
- Connect Web3 wallets
- Explore an immersive open world
- Compete in future multiplayer events

## 🌐 Features
- Open-world gameplay
- Cyberpunk Gulf city
- Mobile-friendly controls
- Web3 wallet integration
- DAF1 in-game economy
- Garage & upgrades
- Nitro & drift system
- Future NFT support

## ⚡ Future Plans
